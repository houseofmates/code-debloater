# Contributing to BloatHunter

First off, thank you for considering contributing to BloatHunter! It’s people like you that help make open-source tools better for everyone.

## Getting Started
1. **Fork the repo** and clone it to your machine.
2. **Install dependencies:** `npm install`
3. **Run locally:** Make sure you have Ollama running, then test your changes with `npm run dev`.

## How to Report a Bug
If you find a bug, please open an issue and include:
- A clear description of the issue.
- Steps to reproduce it.
- Your OS and Node.js version.

## How to Suggest a Feature
Have an idea for a better AST-parsing rule or a new Ollama integration? Open an issue with the tag `enhancement` and tell me what you're thinking!

## Pull Requests
- Please make sure your code follows the existing style.
- If you're fixing a bug, please link the issue you're solving in the PR description.
- Keep PRs focused on one change at a time.

## Code of Conduct
Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
